# CodeAnt AI - Assignment

**## Run Locally**

Clone the project

```bash

git clone https://github.com/KuntalRathod/CodeAnt_AI_Assignment.git

```

Go to the project directory

```bash

cd my-project

```

Install dependencies

```bash

npm install

```

Start the server

```bash

npm run dev

```

Open the application in browser:

```bash

http://localhost:5173

```
